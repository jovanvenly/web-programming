Satuan Unit CSS px, em, & rem

CSS -em-

  1. kalau em, memiliki font size yang static
  2. ukurannya akan menyesuaikan font size pada browser
  3. em to px 1 em = 16px
  4. size 'em' akan inherith jika ada lebih dari satu 'em', font-size akan bertambah

CSS -px-

  1. ukurannya akurat
  2. pixel tidak terpengaruh oleh elemen lain
  3. sering digunakan

CSS -rem-

  1. sifatnya dinamis
  2. rem tidak inherith
  3. ukurannya sama dengan 'em'
